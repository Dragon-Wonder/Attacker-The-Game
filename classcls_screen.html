void clsScreen::start ( void  )

The default constructor for the SDL screen it will try start SDL, and create and window and a renderer, then try to load the textures it will need, if any of these fail it will set bln_SDL_Started to false and return void, when main in main.cpp checks bln_SDL_Started and ends the entire program will it is false. If, however, bln_SDL_Started is true it will continue on with the rest of the program.
